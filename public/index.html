Architecture

Agent 내부 구조

프론트엔드는 FastAPI와 통신하고, 백엔드는 LangGraph StateGraph를 통해 Tool, RAG, Memory, Middleware, OutputParser를 연결합니다.

01

LangGraph

StateGraph와 conditional edge로 진단 흐름을 분기합니다.

02

Tools

장애 유형 분류, RAG 검색, 명령어 추천 Tool을 실행합니다.

03

RAG

Markdown 기반 네트워크 트러블슈팅 문서를 검색합니다.

04

Memory

세션별 대화 이력을 유지해 후속 질문을 처리합니다.

05

Middleware

Tool 호출, LLM 호출, 에러, 실행 완료 로그를 기록합니다.

06

OutputParser

Pydantic 구조로 진단 결과를 JSON 형태로 반환합니다.
